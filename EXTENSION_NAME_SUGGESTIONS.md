## Icon Requirements

### VS Code Marketplace Icon Specifications

**Required Sizes:**
- **128x128 pixels** (PNG format) - Main icon
- **Square format** (1:1 aspect ratio)
- **File format**: PNG with transparency support
- **File size**: Recommended under 100KB

**Design Guidelines:**
- Use simple, recognizable symbols
- Should be readable at small sizes (16x16 in UI)
- Consider using:
  - Linux/Tux penguin silhouette
  - Multiple Linux distribution logos
  - Docker/container symbols
  - Testing/checkmark symbols
  - Code brackets `<>` or `{}`

**Color Scheme Suggestions:**
- Use vibrant but professional colors
- Consider: Blue (Docker), Green (Linux), Orange (testing)
- Ensure good contrast for visibility

**Where Icon is Used:**
- Extension marketplace listing
- VS Code Extensions panel
- Command palette (if applicable)
- Status bar (if applicable)

### Example Icon Ideas

1. **Multiple Linux logos** arranged in a circle/grid
2. **Docker whale** with Linux symbols
3. **Test tube/beaker** with Linux symbols
